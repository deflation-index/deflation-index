# IRENA Sources

This folder contains reports from the International Renewable Energy Agency (IRENA).

## Required Files

| Filename | Description | Download From |
|----------|-------------|---------------|
| `IRENA_renewable_power_costs_2024.pdf` | Main 2024 cost report | [IRENA](https://www.irena.org/Publications) |
| `IRENA_renewable_power_costs_2023.pdf` | 2023 cost report | [IRENA](https://www.irena.org/Publications) |

## Key Data Points

- Solar PV LCOE trends (2010-2024)
- Wind power cost declines
- Global renewable capacity additions

## Citation Format

> IRENA (2025), Renewable power generation costs in 2024, International Renewable Energy Agency, Abu Dhabi.
