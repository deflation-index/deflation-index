# Data Files

This directory contains the source data and calculations for The Deflation Index.

## Files

### `master_deflation_index.xlsx`
**The main composite index combining all three sectors.**

Contains:
- Annual Deflation Index (DI) values (1990-2024)
- M2 money supply growth rates
- Official CPI inflation rates
- Purchasing power calculations
- Gap analysis (stolen abundance)
- Summary statistics

Key outputs:
- Average annual DI: ~-22%
- Tech purchasing power (2024): ~1,500
- Actual purchasing power (2024): ~55
- Stolen abundance: ~90%

### `computing_deflation_index.xlsx`
**Computing sector analysis (35% of DI)**

Tracks:
- Cost per GFLOPS (40% weight)
- Cost per GB storage (40% weight)
- Smartphone performance-adjusted pricing (20% weight)

Key finding: 99.9%+ cumulative deflation

### `communications_deflation_index.xlsx`
**Communications sector analysis (30% of DI)**

Tracks:
- Long distance calling costs (30% weight)
- Mobile data pricing (40% weight)
- Video entertainment costs (30% weight)

Key finding: 100% deflation (free calling via internet)

### `energy_deflation_index.xlsx`
**Energy sector analysis (35% of DI)**

Tracks:
- Solar panel costs (40% weight)
- Lithium-ion battery costs (35% weight)
- LED efficiency and pricing (25% weight)

Key finding: 85-98% cumulative deflation

## Data Quality

✅ **All calculations formula-driven** (no hardcoded values)  
✅ **Sources cited** in each spreadsheet  
✅ **Conservative estimates** (likely understates true deflation)  
✅ **Publicly verifiable** (all source data available)
